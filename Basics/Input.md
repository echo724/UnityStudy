# Input System in Unity

```C#
void Update(){
    Input.anyKeyDown
}
```

## `anyKeyDown`
- When Key is pressed -> True

## `anyKey`
- When Key is being presssed -> True

## `GetKeyDown`,`GetKey`,`GetKeyUp`
- using with `KeyCode`
    - Return Key

## `GetMouseButtonDown`, `GetMouseButton`,`GetMouseButtonUp`
- Return true or false when Mouse is clicked
